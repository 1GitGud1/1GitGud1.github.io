REM

Project Type Professional

Software Used .NET

Languages Used C#, LINQ

Primary Role(s) Lead Programmer

REM (Real Estate Manager) is a desktop application I developed for RBA Investment FZ to streamline property and facility management operations. The system centralizes key tasks such as managing tenant contracts, recording payments, scheduling services, and tracking preventive maintenance activities. Its primary goal is to reduce manual effort, improve accuracy, and provide a reliable platform for day-to-day building management. It was built using the .NET framework, with the front-end developed in Windows Forms, and the database implemented with Entity Framework Core and SQL Server.
