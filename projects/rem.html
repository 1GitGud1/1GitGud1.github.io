REM (Real Estate Manager) is a desktop application I developed for RBA Investment FZ to streamline property and facility management operations. The system centralizes key tasks such as managing tenant contracts, recording payments, scheduling services, and tracking preventive maintenance activities. Its primary goal is to reduce manual effort, improve accuracy, and provide a reliable platform for day-to-day building management. It was built using the .NET framework, with the front-end developed in Windows Forms, and the database implemented with Entity Framework Core and SQL Server.

Entity Framework Core & Database Design

When building the application, I wanted a clean separation between the business logic/UI layer and the data access layer. For this, I created a dedicated class library project that contained all Entity Framework Core models, the DbContext, and migrations. The main Windows Forms application only references this library, which keeps the architecture modular and easier to maintain.

The backend was developed entirely using EF Core which has its own language, LINQ, which then translates into queries of the database language of your choosing which makes it simple to work with any database with almost no knowledge of its querying language. The entity classes and their relationships were written in LINQ and theres one particularly interesting one. 


namespace EFDataAccessLibrary.Models
{
    [PrimaryKey(nameof(BuildingNumber), nameof(ApartmentNumber))]
    public class Apartment
    {
        [DatabaseGenerated(DatabaseGeneratedOption.None)]
        public int BuildingNumber { get; set; }

        [DatabaseGenerated(DatabaseGeneratedOption.None)]
        public int ApartmentNumber { get; set; }

        [Required]
        [MaxLength(50)]
        public string TenantName { get; set; }

        [Required]
        [Column(TypeName = "date")]
        public DateTime ContractStartDate { get; set; }

        [Required]
        [Column(TypeName = "date")]
        public DateTime ContractEndDate { get; set; }

        public bool Ejari { get; set; }

        [Required]
        [MaxLength(50)]
        public int Deposit { get; set; }

        //Amount
        [Required]
        [MaxLength(50)]
        public int Amount { get; set; }

        public List<ApartmentService> ApartmentServices { get; set; } = new List<ApartmentService>();
        public List<ApartmentCheque> ApartmentCheques { get; set; } = new List<ApartmentCheque>();

        public ApartmentPPM? PPM { get; set; }
    }
}
