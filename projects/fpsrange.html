FPS Range

Project Type      Personal
Software Used     Unreal
Languages Used    C++, Blueprints
Primary Role(s)   Gameplay Programmer

FPS Range is a First Person Shooter mini-game that lets you practice your aim on a custom-made sandbox shooting range map. It features basic First-Person Shooter mechanics, Camera Shake, Bullet Decals, Dynamic Aim affected by Recoil, Custom Animations. You can try out the game here: 'link'

Introduction

FPS Range is my first game I developed in Unreal Engine with the intent to learn the engine's basics before starting the development of my next big project this time with a team. The idea was to start with a very bare-bones idea and work on it to a polished state. I'm happy with how it turned out but I'm even more thrilled to move on to more exciting enterprises.
